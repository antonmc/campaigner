Campaigner - mobile design

Author: Anton McConville  |  Source: Source and designs stored at Github

A project to re-imagine collecting signitures for petitions and storing them in a simple database.

The mobile design aims to be clean, simple and to the point - to quickly allow a signiture to be captured digitally. Optionally the signer can send a photo, tweet or communicate support via other digital means.
